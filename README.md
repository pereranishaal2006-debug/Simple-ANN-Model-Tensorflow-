# Simple-ANN-Model-Tensorflow-
This project uses a simple Artificial Neural Network (ANN) built with TensorFlow and Keras. The model learns patterns from data to make predictions. Trained weights are saved in weights.h5 for reuse without retraining.

Artificial Neural Network (ANN) – TensorFlow
Description

This project implements a simple Artificial Neural Network (ANN) using TensorFlow and Keras. The model learns patterns from input data and is used to make predictions.

Files
model.py / model.ipynb – ANN model implementation
weights.h5 – Saved trained model weights
README.md – Project description
Requirements
Python 3.x
TensorFlow
NumPy
Usage
Build the ANN model architecture.
Load the saved weights using weights.h5.
Use the model to make predictions or continue training.
